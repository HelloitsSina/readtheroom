# ReadTheRoom — AI Workplace Prep Skill

Know the room before you walk in.

ReadTheRoom is a Claude skill that helps you rehearse important
workplace conversations before they happen. Paste your proposal,
pitch, or situation — six AI personas respond simultaneously from
different stakeholder perspectives.

Built for people working in Australian workplaces, especially those
from East Asian backgrounds navigating cultural communication norms.

## What it does

- Simulates how 6 different stakeholders would respond to your proposal
- Lets you build custom personas from real context (emails, meeting
  notes, LinkedIn profiles)
- Gives structured, scannable feedback, not generic advice
- Includes a Culture Coach persona that translates Australian workplace
  communication norms for people from East Asian backgrounds

## How to install

1. Open the file `prompt/main.md` in this repository
2. Copy the entire contents
3. In Claude.ai, go to Settings and create a new Project
4. Paste the contents as the Project custom instructions
5. Start a new conversation and type `/readtheroom` to begin

## How to use

Type `/readtheroom` in any Claude conversation to start.

The skill will guide you through two options:
- Option A: Paste your proposal and run the simulation immediately
- Option B: Build custom personas from real context first, then run
